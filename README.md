\# Health Metrics Tracker (HMT)



A full-stack application for tracking and visualizing health indicators across healthcare facilities.



\## Overview



This project demonstrates a complete health information system similar to DHIS2's core functionality, built with modern technologies.



\## Tech Stack



\*\*Backend:\*\*

\- Java 21

\- Spring Boot 3.x

\- PostgreSQL 18

\- Maven



\*\*Frontend:\*\*

\- React 18

\- Material-UI

\- Recharts for data visualization

\- Axios for API calls



\## Project Structure

```

health-metrics-tracker/

├── backend/          # Spring Boot REST API

├── frontend/         # React application

├── docs/            # Documentation and guides

└── README.md        # This file

```



\## Features (Planned)



\- Facility management

\- Health indicator tracking

\- Data entry and validation

\- Real-time dashboards

\- Regional aggregation

\- Data export (CSV)
